# BIST Portfolio Tracker
 
BIST hisselerini takip etmek için geliştirilmiş masaüstü uygulaması.  
A desktop app for tracking BIST stocks and managing your portfolio.
 
---
 
## Özellikler / Features
 
- 537 BIST hissesi arasından arama ve seçim / Search and select from 537 BIST stocks
- Alış fiyatı, adet ve hedef satış fiyatı girişi / Entry for purchase price, quantity and target price
- Anlık fiyat güncellemesi (her 1 dakikada) / Live price updates every minute
- Kar/zarar hesaplama (TL ve %) / Profit/loss calculation (TL and %)
- Hedef fiyata göre doğruluk oranı / Accuracy rate based on target price
- Veriler otomatik kaydedilir / Data saved automatically
 
---

`Python` · `PyQt5` · `yfinance`
